# Money Guy's How Much Should You Save HTML Visualizer

An interactive, single-file web app replicating the Money Guy Show's ["How Much Should You Save?"](https://moneyguy.com/resource/how-much-to-save/) matrix ([video](https://www.youtube.com/watch?v=JHv0ISsl7K0)).

Adds the ability to set your current income + asset holdings to adjust the target % accordingly.

> **Disclosure:** This was generated with Gemini 3.1 Pro.
